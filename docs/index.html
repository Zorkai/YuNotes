What's inside

Everything you need to think on paper — without the paper.

Handwriting, highlighting, shapes, and full PDF markup, tuned for a stylus first and a mouse second.

Ink that doesn't lag

Pressure-sensitive pen, highlighter, and eraser with GPU-accelerated live strokes — the ink stays under the nib, even mid-page at high zoom.

Hold-to-snap shapes

Sketch a rough rectangle, ellipse, line, or triangle, hold still for a beat, and it snaps to a clean shape you can nudge into place.

Real PDF annotation

Open a PDF and write anywhere, re-sharpening the page at any zoom. Your edits reopen fully editable; other apps see a clean flattened copy.

Templates & page extension

Dot grid, lines, or blank — set per page. Drag a page's edge to give yourself more room to the left or right, mid-note.

Selection that just works

Lasso or box-select ink, then move, resize, rotate, duplicate, or recolor — and drag a selection straight across page breaks.

Light, dark, and yours

A full light and dark theme, plus a toolbar you can move, resize, reorder, and trim down to the tools you actually reach for.

Never lose a stroke

Autosave that waits for your pauses, sixty steps of undo, and crash-resilient saves — so a slip never costs the page.

Tuned for the stylus

Palm rejection, remappable pen barrel buttons, and adjustable pressure response — set it up the way your hand works.

Get it running

Installed in about a minute.

Download the release, run one file, and YuNotes lands in your Start menu like any other app.

STEP 01

Download the release

Grab the latest YuNotes-x64.zip from the GitHub releases page.

STEP 02

Run the installer

Unzip it and double-click Install.bat, then click Yes on the prompt.

STEP 03

Open YuNotes

Find it in the Start menu, pin it, and start writing. Uninstalls from Settings like normal.

Two ways to install: the easiest is the Microsoft Store — signed by Microsoft, one-click install and automatic updates, no trust step. Prefer a portable build? The GitHub release is signed with a developer certificate, so Windows asks you to trust it once (the steps above).
